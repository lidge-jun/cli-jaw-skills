## 실전 벤치마크 — 44분 거시경제학 강의

### 테스트 환경

| 항목 | 값 |
|------|---|
| 모델 | Gemini 3.1 Flash-Lite Preview |
| 오디오 | 44분 거시경제학 강의 (MACRO ECON.m4a, 22MB) |
| 슬라이드 | topic01.pdf (Ch.3 The Goods Market, 850KB) |
| Input tokens | ~86,000 (오디오 66K + PDF 20K) |

### 구성별 비교

| 구성 | 시간 | 출력 크기 | Out 토큰 | KaTeX | 구조 | 커버리지 |
|------|------|----------|----------|-------|------|----------|
| PDF + 상세 + Non-essential 마킹 | **32s** | **18KB** | 4,772 | ✅ | 슬라이드별 ✅ | **100% + 비핵심 요약** 🏆 |
| PDF + 오디오 + 상세 프롬프트 | 20.4s | 13KB | 3,826 | ✅ | 슬라이드별 ✅ | 100% |
| PDF + 오디오 + 기본 프롬프트 | 28s | 11KB | 3,226 | ✅ | 슬라이드별 ✅ | 100% |
| 오디오 + 프롬프트 (PDF 없음) | 11.6s | 1.2KB | 357 | ✅ | 요약형 | 75% (FOCUS 누락) |
| 오디오만 (프롬프트 없음) | 181s | 283KB | 65,532 | ❌ | 없음 | ❌ 루프 발생 |
| 필러 마킹 모드 | 161s | 297KB | 65,532 | ❌ | 없음 | ❌ 루프 발생 |

### 핵심 발견

- **PDF가 압축기**: PDF 있을 때 출력 토큰 22배 감소 (65K → 3K), 속도 9배 빠름
- **프롬프트가 품질기**: 상세 프롬프트 → 보강노트(*Supplementary*) 자동 추가
- **필러 마킹 = 안티패턴**: verbatim 모드로 전환, 구조화 능력 상실
- **비용**: 44분 강의 1회 = ~$0.03 (36원) — Clova Note 무료 제한 대비 무의미한 비용
