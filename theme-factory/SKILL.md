# Theme Factory

Apply consistent font and color themes to artifacts (slides, docs, reports, HTML pages).

## Usage

1. Show `theme-showcase.pdf` so users can see all themes visually.
2. Ask which theme to apply.
3. Wait for explicit selection.
4. Read the theme file from `themes/` and apply colors/fonts consistently. Ensure proper contrast and readability.

## Available themes (see `theme-showcase.pdf`)

1. **Ocean Depths** — professional maritime
2. **Sunset Boulevard** — warm, vibrant
3. **Forest Canopy** — natural earth tones
4. **Modern Minimalist** — clean grayscale
5. **Golden Hour** — rich autumnal
6. **Arctic Frost** — cool, crisp
7. **Desert Rose** — soft dusty tones
8. **Tech Innovation** — bold tech aesthetic
9. **Botanical Garden** — fresh organic
10. **Midnight Galaxy** — dramatic cosmic

Each theme in `themes/` includes hex color palettes and font pairings for headers/body.

## Custom themes

When no preset fits, generate a custom theme: choose colors/fonts from any description provided, give it a descriptive name, show for review, then apply.
